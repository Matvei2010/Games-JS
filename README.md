# Games-JS
🌍 World Dev Games — JS Game Simulator
Браузерный симулятор игр на чистом JavaScript
🎮 Играй где угодно: на ПК или смартфоне. Без скачиваний, без установки — только открыл браузер и погрузился в игру.

📌 О проекте
World Dev Games — это платформа-симулятор, объединяющая коллекцию HTML5/JS игр с единым адаптивным интерфейсом. Проект создан для того, чтобы продемонстрировать возможности современных веб-технологий в игровой индустрии и предоставить пользователям простой доступ к качественным мини-играм на любом устройстве.

Ключевые особенности
✅ Кроссплатформенность — игры работают на Windows, macOS, Linux, iOS и Android

✅ Адаптивный интерфейс — автоматическая подстройка под экран ПК, планшета или телефона

✅ Чистый JavaScript — без фреймворков, быстрая загрузка и отзывчивость

✅ Регулярные обновления — ежемесячное пополнение библиотеки

Каждая игра написана на ванильном JavaScript, CSS и HTML5 Canvas.


Адаптивность: Flexbox, Grid, медиа-запросы

Управление: Обработка событий клавиатуры (keydown) и тач-событий (touchstart, touchmove)

Графика: Canvas 2D Context, в будущем WebGL для сложных проектов

📱 Адаптивность
Проект разработан по принципу Mobile First:

На ПК — полноценное управление с клавиатуры, отображение подсказок клавиш

На телефоне — сенсорные кнопки (кастомные или нативные), отключение нерелевантных клавиатурных подсказок

На планшете — гибридный режим: крупные элементы для касания + опциональная клавиатура

🧪 Как запустить локально
Клонируйте репозиторий:

bash
git clone https://github.com/ваш-аккаунт/world-dev-games.git
Перейдите в папку проекта:

bash
cd world-dev-games
Откройте index.html в браузере (рекомендуется использовать Live Server для корректной работы модулей).

Или просто разверните на любом статическом хостинге (GitHub Pages, Netlify, Vercel).
